# Ames Housing Truth: 都市環境と居住価値の相関分析

## このプロジェクトについて
世界中で分析される「Ames Housing Dataset」を、機械学習のモデル構築のためだけでなく、**「都市工学的な視点」と「社会歴史的な考察」**を加えた多角的な調査プロジェクトです。

単なる価格予測の精度を競うのではなく、**「データから見える街のリアル」**を紐解くことを目的としています。

## 主な分析テーマ（全24ファイル）
機械学習のコードの裏側にある、以下のような「暮らしの真実」を追究しました：

* **都市の動脈と負荷:** 幹線道路（Artery）がもたらす渋滞・排気ガスと価格の関係。
* **空間のゆとり:** 大型車社会におけるLotFrontageやGarageCarsの経済的価値。
* **安全の設計:** 袋小路（Cul-de-sac）が持つ居住環境のレジリエンス評価。
* **歴史的考察:** 25年ルール以前の、大型車が支配的だった時代のアメリカの住環境。

## なぜこれをやったのか？
データセットの数字を眺めているうちに、「なぜこの立地はこれほど評価されるのか？」という疑問が次々と湧き上がったためです。各ファイルは、私が興味の赴くままに仮説を検証し、分析を詰め込んだ「研究室の記録」です。
